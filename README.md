# 🌦️ React Weather App
A sleek and responsive weather application built with **React.js** that provides real-time weather information for any city worldwide.

---
## ✨ Features
 Search weather by city name  
 Real-time weather updates  
 Temperature, humidity & wind speed  
 Dynamic weather icons  
 Responsive design for all devices  
 Fast and user-friendly UI  
 
---
## Built With
- ⚛️ React.js
- 🎨 CSS
- 🌐 OpenWeather API
- 📡 Fetch API
